Big-Oh terminology  Big-Oh notation allows us to state that a function of  𝑛 is less than or  equal to another function, up to a constant factor ( 𝑐 ), as  𝑛 grows  toward infinity (asymptotically)  This allows us to characterize the execution time required for a  function in the worst-case scenario\!  We often say:  “ 𝑓(𝑛) is **big-Oh **of  𝑔(𝑛) ”  “ 𝑓(𝑛) is **order** 𝑔(𝑛) ”  And write it as:  𝑓 𝑛 ∈ 𝑂(𝑔 𝑛 ) 12 CSC 225: Algorithms and Data Structures I -- Anthony Estey
